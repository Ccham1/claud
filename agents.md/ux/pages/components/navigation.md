# Navigation Component

## Description
Sticky top navigation bar used across all pages of Amigos30Plus:
- Links: Home, About, How It Works, Chat, Events, FAQ, Contact
- Collapses into hamburger menu on mobile
- Sticks to the top when scrolling

![Navigation Screenshot](../../mocks/navigation.png)

## Acceptance Criteria (ACs)
- Nav bar remains at top of viewport on scroll
- Links scroll smoothly to corresponding sections
- Active section link highlighted in navbar
- Mobile: hamburger menu appears <768px width
- Hover/focus states are accessible
- Nav has sufficient z-index to overlay page content

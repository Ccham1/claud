# homepage hero Component

## Description
A hero banner for the home page.
- Hero slider
- Full width image
- Has html text overlay
- Has a call to action
- rotation is set at 5 seconds
- each item in the slider is clickable

## Acceptance Criteria (ACs)
- Hero slider can accept multple items
- Hero slider must accept html text 
- Speed of slider is every 5 seconds
- call to action button must be clickable
- the banner must sit below the header on page load

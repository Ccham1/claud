# Frontend Tech Stack - Amigos30Plus (Vue 3)

## Framework
- Vue 3 (Composition API)

## Styling
- Tailwind CSS for layout and utility classes
- Optional: SCSS for custom styles

## State Management
- Pinia

## Routing
- Vue Router 4

## Forms & Validation
- VeeValidate for form validation

## Data Fetching
- REST API
- Use Fetch

## Accessibility & SEO
- Semantic HTML
- ARIA roles for nav, buttons, modals
- WCAG 2.1 compliance

## Testing
- Unit: Vitest
- E2E: Playwright

## Build & Deployment
- Build: Vite

## Folder Structure (Vue 3)
Use services/composeables/component structure
